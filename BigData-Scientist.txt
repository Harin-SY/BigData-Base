데이터 관련 직무란?
1. 데이터 분석가 -> "과정"에서 1번부터 4번까지의 업무
2. 데이터 엔지니어 ->"과정"에서 5번째의 업무
3. 데이터 사이언티스트

<데이터 분석 절차>
* 과정
1. 분석 목적 및 방향 결정 - 주제 선정
2. 데이터 선택 및 수집 - 데이터 이해
3. 데이터 탐색 및 정제
4. 모델 생성 및 평가
5. 적용

*업무
1) 인터뷰 및 현황 분석 -> 분석 주제 발굴 -> 분석 방향 정의
2) 가설 수립 -> 분석 데이터 정의 -> 분석 데이터 수집(엄청난 크기의 데이터를 다룸)
3) 샘플링 및 데이터 정제(전수데이터를 한 번에 분석하기) -> 파생 변수 생성 -> 데이터 탐색(python등 분산처리 기능 사용) 
4) 데이터 모델링(기계학습, 딥러닝 사용) -> 결과 해석 -> 모델 평가(실시간 분석)
5) 프로세스 설계 -> 시스템 개발(빅데이터 플랫폼 사용) -> 운영적용 및 모니터링(시각화)

* 4번의 결과가 무의미 하다면 2번으로 다시 돌아가 점검을 해야 한다.

<데이터 사이언티스트의 주요 역랑>
 - 비즈니스 지식
 - 통계학 및 기계학습
 - Python 등의 프로그래밍 기술
 - Big Data 기술의 이해
 - 시각화 및 보고
 - 다양한 방법을 시도하는 호기심

척도 : 척도에 따라 분석 방법이 달라진다.
  1) 포괄성의 원칙 : 척도 안의 모든 경우의 수를 포함 시켜야 한다.
  2) 상호 배타성의 원칙 : 척도 안에 중복되는 경우의 수가 없어야 한다.

명목 척도 : 측정되는 항목이 상호 배타적인 특성만을 가진 척도이다.(이산형 변수/비연속형 변수)
              -> 남/여

서열 척도 : 명목 척도들 중 서열이나 순위가 존재하는 척도이다. (이산형 변수/비연속형 변수)
             -> 학급 석차

등간 척도 : 서열 척도들 중 항목들의 간격이 일정한 척도(연속형 변수)
             -> 상, 중, 하

비율 척도 : 등간 척도 중 아무것도 없는 상태를 "0"으로 정할 수 있는 척도(연속형 변수)
             -> 매출액


도수 분포표 : 수집된 자료를 쉽게 이해할 수 있도록 정리된 표
                   -> 특정 항목 또는 특정 범위에 속하는 빈도수를 나타낸 표

막대 래프 : 비연속형 변수(서열 척도, 명목 척도)에 사용되는 그래프로
                  각 항목에 속하는 빈도수를 나타낸 그래프
                  -> 도수 분포표를 좀 더 쉽게 표현할 수 있다.

히스토그램 : 연속형 변수(서열 척도, 등간 척도)에 사용되는 그래프로
                 일정 범위에 속하는 빈도수를 나타낸 그래프

선 그래프 : 히스토그램의 끝부분을 선으로 연결한 그래프


공분산 : 두 변수가 평균으로부터 멀어지는 정도
            -> 한 변수가 자신의 평균으로부터 멀어질 때 다른 변수가 자신의 평균으로부터 멀어지는 정도를 의미
               
상관계수 : 두 변수 간의 관계를 나타냄, 하나의 변수가 변화함에 따라 다른 변수의 변화 정도를 의미
